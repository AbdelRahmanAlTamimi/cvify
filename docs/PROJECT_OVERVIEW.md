# Project Overview

## What is Cvify?

Cvify is a self-hosted web application that helps you create and manage professional CVs using AI. It tailors your CV to match specific job descriptions automatically.

## How It Works

1. **Create a Profile** - Store your professional information (experience, education, skills, etc.)
2. **Generate CV** - Paste a job description and AI creates a customized CV
3. **Download** - Get your tailored CV as a PDF file

## Key Features

- **AI-Powered** - Uses Groq AI to match your profile with job requirements
- **Profile Management** - Store multiple profiles with different information
- **CV History** - Keep track of all generated CVs
- **PDF Generation** - Professional PDF output
- **Self-Hosted** - Your data stays on your server
- **Web Interface** - Easy-to-use browser interface

## Tech Stack

- **Backend**: NestJS (Node.js framework)
- **Frontend**: React + Vite
- **Styling**: Tailwind CSS + Shadcn UI
- **Runtime**: Bun (fast JavaScript runtime)
- **Database**: PostgreSQL
- **ORM**: Prisma
- **AI**: Groq API (LLaMA model)
- **PDF**: PDFMake
- **Deployment**: Docker

## Project Type

This is a **small to medium-sized project** with:

- ~1,500 lines of code
- 4 main modules (Profiles, CVs, Groq, Prisma)
- Simple architecture
- Single developer-friendly
- Easy to understand and modify
